# Euro-jobs-careers
website with GSAP animations for Euro jobs careers a recruitment agency for European jobs

Concept: Focuses on the "Path" aspect with fluid motion. A visual metaphor for smooth career transitions across borders.
Visuals: interconnected lines, map-like abstract patterns, and a sophisticated European color palette (deep flag blue, bright yellow accents, slate grey).
GSAP Usage: Kinetic typography (text moving on scroll), horizontal scrolling sections for featured jobs, and dynamic cursor effects.
